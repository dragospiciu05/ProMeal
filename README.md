ProMeal este un sistem de gestionare bazat pe Python care automatizează planificarea meselor zilnice. 
Aplicația folosește arhitectură modulară (OOP)  și o bază de date relațională.

Arhitectură OOP: Cod sursă structurat pe obiecte (User, Meal, Manager) pentru o mentenanță ușoară.

Sistem de Autentificare: Managementul utilizatorilor.

Algoritm de Diversitate: Interogări SQL complexe care previn duplicarea aceleiași mese într-un interval de 24 de ore.

Determinarea automată a indicelui BMI (Body Mass Index)

Persistența Datelor: Integrare completă cu SQLite pentru salvarea profilului, jurnalelor alimentare și a meselor favorite.

În cadrul fișierelor este inclus test_client.py, o interfață de tip CLI destinată testării funcționalităților aplicației, precum și creării și populării bazei de date..
